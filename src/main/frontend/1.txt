가제 : Cast Commuinity(CC)
(짭브리타임) 

개요 : 에버랜드 혹은 롯데월드 직원들은 서로 소통할 수 있는 방법이 많이 없음
많은 인원들임에도 불구하고 소통 수단이 없다는 것에 주목, 여러 정보를 주고 받고
친목 도모를 장려하기 위해 해당 어플 및 웹사이트를 생성하려고 함

기능 : 
- 게시판 기능이 약 90%
(자유게시판,장터,분실물 등등)
- 로그인/회원가입
- 파크 내 각종 정보(버스 시간표, 밥, 파크 일정) 전달
- 쪽지 기능
- 익명 혹은 비익명 지정 가능 기능
- 내 정보 변경 가능 기능
- 달력 몇시간 일했는지, 월급 예상 기능


언어 :
미정

+ 소모임과 같이 캐스트들의 동아리를 따로 생성할 수 있는(보류)
+ 파크 내 지도 (보류)
